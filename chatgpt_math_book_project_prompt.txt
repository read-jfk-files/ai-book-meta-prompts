This is a project for generating reader's guide style tutorial and speed-run survey documents about various mathematics topics. the intent is for you to generate a long-form document that teaches the whole topic in order for me to avoid needing to spend hours reading through multiple textbooks and papers containing duplicated material, but instead allow me to quickly get up to speed on a new topic. Please generate all output as an HTML artifact that is a stand-alone single file that i can download and run locally in my web browser. Since all of the topics will be math related, you can use javascript and libraries like MathJax for rendering and it is acceptable to use 3rd party external javascript libraries linked on remote CDN networks. In your generated output about math topics, always try to include several toy examples or canonical problems which are used to demonstrate the concepts, theorems or algorithms. Because i learn better when I can see simple examples and build upon them into larger and larger theories. Also try to mention and explain extreme edge cases, such as pathological problems or toy examples which demonstrate where the theorems or algorithms do not work. This is because when i can see the edge cases where we know the theory breaks down and does not apply, that also helps me learn the material better because i can see the limitations and the unknown territory. When you assemble the information to generate the Reader's Guide, use your knowledge as well as perform a web search of the math literature to find between 3 and 5 of the best or most cited or most impactful papers which are surveys or tutorials of the topic. Then use those survey papers to augment your knowledge of the topic to enhance your level of detail in presenting the topic.
